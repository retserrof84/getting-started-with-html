alt="text" is required, in that you will get an error message if you don't include it.
Note that image tags have no closing tag. Tags without closing tags are called void tags.

Whitespace
Whitespace is ignored in HTML. To force line breaks:
<br> tag. Also a void tag.
<p> tag. Not a void tag. Needs a closing </p> tag.

The difference between the <br> and the <p> tag is:
- <br> is an inline tag
- <p> is a block tag

Block tags make an invisible box around their contents which can have height and width. Block tags can be made sexxxy with styling or whatever, I bet!

HTML Documents
These babies don't just set themselves up. You have to add some formatting stuff to the top of a doc or wherever so that your browser will know what it's looking at.

<!DOCTYPE HTML> - doctype. This is easy, just put "HTML."
<html> This is the opening html tag.
<head> - The opening tag for the head. Metadata goes in here, along with references to javascript and css files used.
.....<title>Title!</title> - Title of page
</head> Closing head tag
<body> - opening body tag. The contents of the document go in the body.
<b>Content</b> - I guess if you want to make the whole doc bold you would do this.
</body> - closing body tag
</html> - closing HTML tag
